Блок 3: Проверка логики валидации

Название: Отсутствие валидации при вводе текста в поле
Ввести в поле ввода email invalid email with spaces
Не нажимая кнопку, убедиться, что сообщение об ошибке не появляется.
Ожидаемый результат: Сообщение об ошибке не отображается. Валидация происходит только по нажатию кнопки.
